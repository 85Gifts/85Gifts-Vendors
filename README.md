85Gifts Vendors App Created with [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## To Get Started

```bash
Create a ".env.local" file at the root of your folder and paste this: 

API_URL=https://eight5giftsvendors-be.onrender.com
NEXT_PUBLIC_API_URL=https://eight5giftsvendors-be.onrender.com

npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
